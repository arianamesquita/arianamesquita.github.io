CSS - Cascading Style Sheets (Folhas de Estilo em Cascata)

 Documentos CSS são empregados para estilizar o conteúdo de um documento HTML, ou seja, controlar a apresentação de documentos de marcação.
 O HTML controla a estrutura de uma página ao passo que o CSS determina sua aparência.
 Uma folha de estilos é uma coleção de regras de formatação aplicadas a um documento HTML.

 As regras CSS podem ser aplicadas de três formas distintas:

  • inline: aplicados diretamente em um elemento HTML.
  • incorporador: O elemento style permite aplicar várias regras de uma vez no próprio documento HTML.
  • folha de estilo externa: um documento com extensão .css permite aplicar regras a um website inteiro de uma vez.

 Os estilos são aplicados na ordem que são encontrados, cascateando estilos externos até os estilos inline.

Sintaxe de uma regra CSS

 Regra CSS é a unidade básica de construção de uma folha de estilos.
 Sintaxe de uma regra:

seletor {
  propriedade: valor;
}

 A regra é composta de um seletor (alvo da regra) e de uma declaração (parâmetros de estilo).
 A declaração, por sua vez, é composta por um par propriedade: valor

Exemplo de uma regra CSS

 Abaixo temos uma regra CSS que aplica três declarações:

p {
 text-align: center;
 color: yellow;
 font-size: 20px;
}

  • Seletor: elemento p(regra aplicada a parágrafos)
  • Declarações: text-align; center; color; yellow; font-size: 20px;
  • Propriedades: text-align, color, font-size.

Tipos de Seletores
  1. Seletor Universal: Representa todos os elementos.
    Simbolizado pelo *.

    2. * {
     color: green;
    }

  2. Seletor de Tipo (ou de Elemento): Representa um determinado elemento.

    3. p {
     color: red;
    }

    h1 {
     color: yellow;
    }

  3. Seletor de Classe: Aplica estilo a elementos com o atributo class definido.

    4. .verde {
     color: green;
    }

    h1.verde {
      color: green;
    }

  4. Seletor de ID: Aplica estilo a elementos com o atributo id definido.

    5. #corAzul {
      color: blue;
    }

    h1#corAzul {
      color: blue;
    }

Convenções para nomes de id e class

 Algumas convenções importantes para seguir ao criar nomes para atributos id e class:

  • Não usar espaços em branco
  • Não empregar caracteres especiais
  • CSS é case-sensitive
