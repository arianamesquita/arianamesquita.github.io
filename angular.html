1. Sobre o Angular

Angular é um framework para densenvolvimento front-end muito popular entre os desenvolvedores, o Angular é mantido pelo Google e foi lançado pela primeira vez em 2009 . É um framework de código aberto que usa TypeScript e JavaScript e é famoso por criar aplicativos de página única.

Sites e aplicativos como eBay e Airbnb, Upwork, Lego, PayPal, Netflix usam o Angular, que reflete o quanto ele é poderoso em construção para escala.

A curva de aprendizado pode ser íngreme, no entanto, e o código às vezes pode ser volumoso para compilar.

1.1 Sobre as versões do Angular

Originalmente foi criado o chamado AngularJS mas depois reformaram inicialmente para retirar o JS do nome e reduzir a confusão que ela estava causando por não ser somente voltada para o JavaScript. A próxima versão lançada foi a Versão 2.0 que trouxe mudanças drásticas e criou algumas controvérsias entre os desenvolvedores.

Foi lançada então a Versão 4.0 ignorando a versão 3 que era incompatível com a 2, a 4 já veio compatível com a versão 2.0. Nela introduziram Solicitações HTTP e novos eventos para Guardas e Resoluções. Em 2017 foi lançada a Versão 5.0 com suporte para aplicativos progressivos e melhorias no Material Design.

Após várias versões e melhorias, a atual como pode ser conferido no site está sendo a Versão 16. Os sites de desenvolvedores falam sobre as grande novidades que esta versão traz.

Para instalar o Angular sem ser alguma versão específica é necessário instalar inicialmente o Node.js, é através dele que a instalação é feita atualmente. O próprio site do aplicativo orienta essa forma de instalação como pode verificar aqui no site oficial.
Vamos iniciar instalando o Node.js então.

3. Instalando o Angular

Lembrando novamente que pode ser conferido no site do Angular caso surjam dúvidas, aqui está o link do Site Oficial já levando direto para onde informa como instalar o Angular. Para instalar o Angular agora pode conferir dentro da sua IDE de escolha, utilizando o console e vai colocar o código:

npm install -g @angular/cli

4. Criando um projeto em Angular

Para criar um novo workspace e iniciar uma aplicação é necessário digitar no console ou barra de comando o seguinte código:

ng new "nome_da_aplicação"

5. Executando aplicações em Angular

Para executar uma aplicação angular o desenvolvedor digita um comando no terminal e digita enter para ele processar, o comando é o seguinte:

ng serve
